## 🧠 Project Overview

A machine learning system that predicts whether a user will click on an online advertisement based on behavioral patterns such as **time spent on site** and **salary**.

This project explores how user engagement can be translated into predictive insights using both **linear** and **ensemble learning techniques**.

---

## ⚡ What This Project Does

1. 📥 Loads and analyzes user interaction dataset
2. 📊 Performs exploratory data analysis (EDA)
3. ⚙️ Preprocesses data using feature scaling
4. 🤖 Trains classification models:

   * Logistic Regression
   * Gradient Boosting
5. 📈 Evaluates model performance using:

   * Confusion Matrix
   * Classification Report
6. 🎯 Visualizes decision boundaries for better interpretability

---

## 🔍 Key Highlights

* Clear distinction between **clicked vs non-clicked users**
* Strong baseline using **Logistic Regression**
* Improved performance with **ensemble methods**
* Visualization-driven understanding of model behavior
